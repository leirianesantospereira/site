1. Componentes de Hardware

1.1 Componentes Internos

1.1.1 CPU

A CPU (Unidade Central de Processamento) é o cérebro do computador.

1.1.2 Armazenamento

Os dispositivos de armazenamento guardam dados de forma permanente.

1.1.3 Memória

A memória RAM armazena dados temporários para acesso rápido.

1.1.4 Fonte de Alimentação

Responsável por converter energia elétrica e alimentar os componentes.

1.2 Componentes Externos

1.2.1 Teclado

Dispositivo de entrada para digitação de caracteres.

1.2.2 Monitor de vídeo

Exibe informações visuais processadas pelo computador.

1.2.3 Mouse

Dispositivo apontador para interação com a interface gráfica.

1.3 Interfaces de Entrada e Saída

1.3.1 Interfaces sem fio

Permitem conexão de dispositivos sem uso de cabos.

2. Grandezas Elétricas

2.1 Potência Elétrica

Mede a taxa de consumo de energia elétrica.

2.2 Tensão Elétrica

A diferença de potencial entre dois pontos de um circuito.

2.3 Corrente Elétrica

Fluxo de elétrons em um condutor.
